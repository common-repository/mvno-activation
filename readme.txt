== Installation ==
###Before You Start
Here are some things to know before you begin this process.
- You need to download a plugin called mvno-settings from the store before you can actually use it
- You should be signed up with https://mvnop.com/ (MVNOP)
- They will provide you with MVNO number and auth code
- Add that number and auth code into settings page
- Deleting removes the connection between MVNO Activations and Pareteum MVNO Services, and uninstalls the plugin from your site.

== Frequently Asked Questions ==

= What is MVNO ? =

A mobile virtual network operator (MVNO) is a wireless communications services provider that does not own the wireless network infrastructure over which it provides services to its customers. A MVNO enters into a business agreement with a mobile network operator to obtain bulk access to network services at wholesale rates, then sets retail prices independently

= How can you use this Plugin for your MVNO Services? =

You can use this plugin if you are already signed up with pareteum for MVNO services

= Do I need to install some other plugin to use this plugin? =

Yes you need to install MVNO Settings plugin before you can actually use MVNO Activations plugin
